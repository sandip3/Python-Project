# 🐍💧🔫 [Snake, Water, Gun Game](./Project1.py)

## Technology
- 🖥️ **Language**: `Python`

## Description
This program simulates the classic "Snake, Water, Gun" game, a variation of Rock, Paper, Scissors.

### Game Rules:
- 🐍 **Snake vs. Water**: Snake drinks the water, hence wins.
- 💧 **Water vs. Gun**: The gun will drown in water, hence a point for water.
- 🔫 **Gun vs. Snake**: Gun will kill the snake and win.

The user will play against the computer, which randomly chooses its option. The game will prompt the user to play again after each round and can display the rules if needed.

### How to Play:
1. 🏁 Run the program.
2. 🐍💧🔫 Choose your option: Snake, Water, or Gun.
3. 🤖 The computer will randomly choose its option.
4. 🏆 The winner will be displayed.
5. 🔁 You will be prompted to play again or exit.

### Features:
- 🎮 Easy to play.
- 🤖 Random choices by the computer.
- ℹ️ Option to display game rules.
